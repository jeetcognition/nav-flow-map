Select a page

The flow reads top to bottom. Click any page node to see how to reach it and the test cases linked to it (Sanity / Regression).

Main flow:
1. Log in (OTP)
2. Landing page → type jeet-test-org in the search bar → click jeet-test-org
3. Top-left corner → click the logo / jeet-test-org ▾
4. Enterprise settings → pick a tab

Toolbar: + Add page adds a node (with optional draft test cases for an LLM to rewrite); Edit in the panel makes fields and test case text editable. Edits persist in this browser (localStorage); Save to repo commits them to GitHub so they're permanent for everyone.
Sources: qa-testing/nav_graph.md · qa-testing/testcases/*.md · Suites: Sanity (SM/SN) & Regression (RG)
